# NewsSentiment
A program designed to scrape the headline of a news website via a link and return the sentiment of it being positive, negative, or neutral

## How to run

### 1. Run in development mode (optional, if you want to run in python)
python news.py

### 2. Build executable (what you should do)
Double-click `build.bat`

### 3. Run the app
Open `dist/news.exe`
